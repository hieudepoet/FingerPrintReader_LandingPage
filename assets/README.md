# Assets Folder

## Hướng Dẫn Thêm File

### 1. Background Image
Đặt file ảnh background vào đây với tên: **background.jpg**
- Kích thước khuyến nghị: 1920x1080 hoặc lớn hơn
- Format: JPG, PNG
- Gam màu: Đỏ/Vàng chủ đạo

### 2. Emblem (Quốc Huy)
Đặt file quốc huy vào đây với tên: **emblem.png**
- Format: PNG (có nền trong suốt)
- Kích thước: 200x200px trở lên

### 3. Audio Files (Tùy chọn)
Nếu muốn có âm thanh, thêm các file:
- **intro-music.mp3** - Nhạc nền intro
- **activation.mp3** - Âm thanh khi chạm nút
- **countdown.mp3** - Âm thanh đếm ngược
- **launch.mp3** - Âm thanh bùng nổ

## Lưu Ý
- Nếu không có file, ứng dụng vẫn chạy bình thường
- Âm thanh sẽ tự động bỏ qua nếu không tìm thấy file
